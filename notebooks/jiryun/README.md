# 웰니스 앱 리텐션 분석 EDA (지륜)

**분석 목표**: 헤이빗 앱의 30일 리텐션 하락 원인 파악을 위한 탐색적 데이터 분석

---

## 데이터 개요

| 파일 | 규모 | 주요 컬럼 |
|------|------|-----------|
| `01_User_Profile.csv` | 12,500명 | User_ID, 가입일자, 가입경로, 기기, 알림수신동의여부 |
| `02_Event_Log.csv` | 1,757,262건 | User_ID, Event_Time, Event_Type, Session_ID, 알림_유형 |

- **분석 기간**: 2025-01-01 ~ 2025-06-30
- **통합 데이터**: event left join user → 1,730,806행 × 11컬럼 (`user_preprocessed.csv`)

---

## 전처리 요약

결측치는 "왜 없는지 원인"을 먼저 파악한 후 처리 방식을 결정했습니다.

| 컬럼 | 처리 방식 | 근거 |
|------|----------|------|
| `Event_Type` 결측 (26,456건) | **행 제거** | 이벤트 유형 없으면 분석 불가 |
| `Session_ID` / `알림_유형` 결측 | `'없음'` 채움 | 알림 이벤트 구조적 결측 — 없는 게 정상 |
| `가입경로` / `기기` 결측 | `'미상'` 채움 | 세그먼트 분석 시 별도 그룹으로 처리 |
| `알림수신동의여부` 결측 (116명) | `False` 채움 | 명시적 동의 기록 없으면 미동의로 간주 |
| `알림수신동의_변경일자` | `datetime64` 변환, `NaT` 유지 | 변경 이력 없는 유저는 구조적 결측 |
| 3/10~14 로그 장애 구간 | `is_log_anomaly` 컬럼으로 플래그 | 제거 대신 플래그 — 분석별 유연한 필터링 |

> **완전 미활동 유저 47명**: 프로필은 있으나 이벤트 로그 없음. 전원 2025년 3월 가입자 (퍼포먼스광고 29명 / 오가닉 17명). 3월 로그 장애와 연관 가능성 있음.

### 팀 논의 필요한 전처리 결정

아래 항목은 팀원 분석 결과와 수치가 달라질 수 있어 기준 통일이 필요합니다.

| 항목 | 현재 처리 | 논의 포인트 |
|------|----------|------------|
| `알림수신동의여부` 결측 → `False` | 보수적 처리 | 알림 효과 분석 시 이 116명 처리 기준 |
| 리텐션 분모 | 이벤트 발생 유저 12,453명 기준 | 전체 가입자 12,500명 기준으로 쓸지 통일 필요 |
| `is_log_anomaly` 제외 여부 | 분석별 선택 | 코호트 리텐션에서는 반드시 제외 권장 |

## EDA 주요 발견

| # | 발견 | 수치 | 시사점 |
|---|------|------|--------|
| 1 | **3/10~14 로그 장애** | 해당 구간 이벤트 70% 누락 | 3월 코호트 수치 신뢰 불가 — 반드시 필터링 |
| 2 | **2월 신규 유입 2배 폭증** | 2,100명 → 4,384명 (+109%) | 광고/오가닉 비율 동일 → 외부 바이럴 추정 |
| 3 | **2월 코호트 D30 전 기간 최고** | 29.1% | 퀄리티 높은 유입 — 성공 요인 역추적 가치 있음 |
| 4 | **5월 이후 이벤트 수 하락** | 14,000건/일 → 하락 추세 | 기존 유저 활동량 감소 시그널 |
| 5 | **온보딩 완료율 45.9%** | 5,719 / 12,453명 | 절반 이상이 온보딩 전 이탈 |
| 6 | **온보딩 완료 시 D30 1.5배** | 완료 29.5% vs 미완료 20.1% | 온보딩이 리텐션 핵심 분기점 |
| 7 | **앱실행 30회가 이탈 임계점** | 11~30회 구간 3,688명 → 31~50회 808명으로 급감 | 30회 전후에서 습관화 성공 여부 갈림 |
| 8 | **알림 오픈율 10.9%** | 21,219 / 194,324건 | 알림 콘텐츠·타이밍 개선 필요 |

|------|------|

### 앱실행 횟수 분포 (유저 활동 양극화)

|------|--------|
| 1회 | 290명 |

## 제안

- **온보딩 퍼널 분석 우선**: 완료 시 D30이 1.5배 높고, 완료율이 45.9%뿐. 어느 단계에서 이탈하는지 파악하면 즉시 개선 포인트 도출 가능.
- **30회 전후 넛지 실험**: 11~30회 유저(3,688명)가 가장 많지만 그다음 구간으로 넘어가지 못하고 이탈. 이 시점에 챌린지 추천·푸시 등 리텐션 넛지 실험 검토.
- **2월 코호트 역추적**: D30 29.1%로 전 기간 최고. 이 유저들의 온보딩 완료율·기능 사용 패턴을 분석하면 장기 잔류 요인 파악 가능.
- **알림 효과 검증**: 알림 동의 64.5%인데 오픈율은 10.9%. 동의 유저 vs 미동의 유저 리텐션 비교로 알림이 실제로 효과 있는지 먼저 확인 필요.
