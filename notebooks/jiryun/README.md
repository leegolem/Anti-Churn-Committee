## 2차 EDA (`2eda.ipynb`)

### 분석 목적

1차 EDA에서 확인된 온보딩 완료율(45.8%)과 리텐션 격차를 심층 분석. 온보딩 완료 여부가 D30 리텐션의 핵심 분기점인지 검증하고 개선 방향 도출.

### 1. 온보딩 미완료 유저 행동 패턴

| 항목 | 수치 |
|------|------|
| 미완료 유저 수 | 6,781명 |
| 미완료 유저 평균 이벤트 수 | 122건 |
| 미완료 유저 중앙값 이벤트 수 | 43건 |
| 이벤트 1~3건 유저 | 251명 (3.7%) |

미완료 유저도 수면기록·운동기록·식단기록·챌린지 등 앱 핵심 기능을 전부 사용하고 있음.

> **결론**: 온보딩이 선택적 구조이기 때문에 완료율이 낮은 것. 앱 흥미 부재가 원인이 아님.

### 2. 온보딩 골든타임

완료 유저 5,719명 전원이 **가입 당일 24시간 이내** 온보딩 완료. 당일을 넘기면 완료 확률 0%에 수렴.

| 완료 유저 `onboarding_elapsed_hours` | 수치 |
|--------------------------------------|------|
| 최솟값 | 6.98시간 |
| 중앙값 | 14.72시간 |
| 최댓값 | 23.01시간 |

> **전략 제안**: 가입 후 24시간 이내 온보딩 완료 유도 푸시알림 또는 인앱 메시지.

### 3. 온보딩 완료 여부별 D30 리텐션

| 구분 | 전체 유저 수 | D30 유저 수 | D30 리텐션 |
|------|------------|------------|-----------|
| 온보딩 완료 | 5,716명 | 1,685명 | **29.5%** |
| 온보딩 미완료 | 6,731명 | 1,352명 | 20.1% |

완료 유저 D30이 미완료 대비 **1.5배** 높음.

> 단, 인과관계 확정을 위해서는 A/B 테스트 필요 (원래 앱에 관심 높은 유저가 온보딩도 완료하고 D30도 높을 가능성).

### 4. 온보딩 완료 시각대별 D30 리텐션

`signup_date`에 시각 정보 없어 `onboarding_elapsed_hours`를 **가입일 자정(00:00) 기준 완료 시각대**로 해석.

| 시각대 | 유저 수 | D30 리텐션 |
|--------|--------|-----------|
| 자정~오전 9시 | 744명 | ~26% |
| 오전 9시~오후 3시 | 2,213명 | **~31%** (최고) |
| 오후 3시~오후 8시 | 1,747명 | ~30% |
| 오후 8시~자정 | 1,015명 | ~28% |
| 미완료 | 6,781명 | 20.1% |

모든 완료 구간이 미완료(20.1%)보다 높음 → **온보딩 완료 자체**가 핵심.

### 5. 세그먼트별 온보딩 완료율

| 세그먼트 | 완료율 |
|----------|-------|
| iOS | 45.6% |
| Android | 46.0% |
| 오가닉 | 45.0% |
| 퍼포먼스광고 | 46.4% |
| 알림 동의 | 44.9% |
| 알림 미동의 | 47.2% |

모든 세그먼트에서 45~47% 수준으로 일정.

> **결론**: 특정 유저 그룹 문제가 아닌 **전체 유저 대상 구조적 문제**. 전체 대상 개선 필요.

### 6. 온보딩 소요시간별 D30 리텐션

앱 첫 실행~온보딩 완료까지 걸린 시간 기준.

| 소요시간 구간 | 유저 수 | D30 리텐션 |
|--------------|--------|-----------|
| 1분 이내 | 2,240명 | 26.2% |
| 1~3분 | 13명 | 7.7% |
| 3~5분 | 10명 | 40.0% |
| 5~10분 | 24명 | 29.2% |
| 10~30분 | 107명 | 29.0% |
| 30분 이상 | 3,262명 | **31.9%** |

> ⚠️ 가상 데이터 한계: 소요시간이 정확히 30초로 고정된 유저가 2,240명(39.6%)으로 신뢰도 낮음. 실제 데이터에서 재검증 필요.

## 제안

- **온보딩 필수화**: 건너뛰어도 앱 이용 가능한 현재 구조 변경 또는 가입 당일 완료 유도 인앱 메시지/푸시 전략
- **30회 전후 넛지**: 11~30회 구간(3,688명) → 31~50회 구간(808명) 이탈 방지 — 해당 시점 챌린지 추천·푸시 실험
- **2월 코호트 역추적**: D30 29.1%로 전 기간 최고 — 이 유저들의 온보딩 완료율·기능 사용 패턴 분석으로 장기 잔류 요인 파악
- **알림 효과 검증**: 알림 동의 64.5%인데 오픈율 10.9% — 동의 vs 미동의 유저 리텐션 비교로 알림 실효성 확인
- **인과관계 검증**: 온보딩 완료 → D30 향상의 인과관계 확정을 위해 A/B 테스트 필요
